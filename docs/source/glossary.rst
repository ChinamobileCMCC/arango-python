********
Glossary
********

.. glossary::
    :sorted:

    Edges REST Api
        REST API for manipulate Edges trought HTTP interface of ArangoDB.
        Documentation about `RestEdge <https://github.com/triAGENS/ArangoDB/wiki/RestEdge>`_

    waitForSync
        This argument may be ``True`` or ``False``. If it's ``True``
        then you'll get response from the server when *Document*,
        *Edge* or *Collection* will be saved on disk.
